# Troubleshooting

This section helps you resolve common issues and optimize your experience with the React Flow Rule Engine.

### "Invalid Connection" Error

**Cause**: Trying to connect incompatible node types

**Solution**: Only connect nodes that are designed to work together

**Example**: Don't connect Initial Node directly to Action Node

**Valid Connections**:

- Initial Node â†’ Rule Group
- Rule Group â†’ Condition Node
- Condition Node â†’ Operator Node
- Operator Node â†’ Action Node

### "Cycle Detection" Error

**Cause**: Circular connections in your flow

**Solution**: Remove the circular connection

**Check**: Look for nodes that connect back to themselves

**Example**: If Node A connects to Node B, and Node B connects back to Node A

### "Disconnected Node" Warning

**Cause**: Node is not connected to the main flow

**Solution**: Either connect the node or remove it

**Note**: Some nodes can be standalone (like Initial Node)

**Check**: Ensure all nodes serve a purpose in your workflow

### Validation Panel Not Showing

**Cause**: Panel might be collapsed

**Solution**: Click the expand arrow in the validation panel header

**Location**: Top-right corner of the interface

**Alternative**: Check if panel is hidden behind other elements

### Minimap Not Visible

**Cause**: Minimap might be hidden or flow is too small

**Solution**: Zoom out or check minimap settings

**Location**: Bottom-right corner of the flow area

**Check**: Ensure flow has enough content to display minimap

### JSON Generation Fails

**Cause**: Flow has validation errors

**Solution**: Fix all validation errors first

**Check**: Ensure all required fields are filled

**Steps**:

1. Check validation panel for errors
2. Fix all red error indicators
3. Ensure all nodes are properly configured
4. Try generating JSON again

### Nodes Not Appearing

**Cause**: Nodes might be outside visible area

**Solution**: Click "Fit View" button or use minimap to navigate

**Prevention**: Use minimap to keep track of node positions

**Check**: Look for nodes in the minimap overview

## ðŸ”§ Performance Issues

### Large Flows

**Symptoms**: Slow performance, laggy interactions

**Solutions**:

- Use **minimap** for navigation
- **Group related nodes** together
- **Save frequently** to avoid data loss
- **Break down** complex flows into smaller parts

**Optimization Tips**:

- Keep node count reasonable (`<` 50 nodes)
- Use groups to organize nodes
- Minimize unnecessary connections
- Regular cleanup of unused nodes

### Complex Rules

**Symptoms**: Difficult to manage, hard to understand

**Solutions**:

- **Break down** complex rules into smaller groups
- **Use operators** to combine conditions logically
- **Validate frequently** to catch errors early
- **Document** your rule logic

**Best Practices**:

- Start simple and add complexity gradually
- Use descriptive names for all elements
- Group related conditions together
- Test incrementally

### Navigation Issues

**Symptoms**: Hard to find nodes, difficult to navigate

**Solutions**:

- **Mouse wheel**: Zoom in/out
- **Space + drag**: Pan around the flow
- **Ctrl + mouse wheel**: Fine zoom control
- **Use minimap**: Click to jump to different areas

**Navigation Tips**:

- Use "Fit View" to see entire flow
- Use minimap for quick navigation
- Organize nodes logically
- Keep related nodes close together
